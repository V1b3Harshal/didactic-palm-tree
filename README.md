# Convis Landing Page

A high-performance, premium landing page for **Convis**, an AI Voice Agent platform. Built with Next.js 15, React 19, and Tailwind CSS.

## 🚀 Features

- **Human-Like AI Demo**: Interactive voice agent simulation.
- **Glassmorphic UI**: Modern, premium design with neumorphic elements.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Contact Form**: Integrated with Nodemailer for lead generation.
- **Performance Optimized**: Using Next.js 15 features and optimized fonts.
- **Animations**: Powered by Framer Motion and GSAP for a smooth experience.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/), [GSAP](https://gsap.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Forms**: [React Hook Form](https://react-hook-form.com/), [Zod](https://zod.dev/)
- **3D Elements**: [Spline](https://spline.design/)

## 📦 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/V1b3Harshal/didactic-palm-tree.git
   cd didactic-palm-tree
   ```

2. **Install dependencies**:
   ```bash
   pnpm install
   ```

3. **Configure Environment Variables**:
   Copy `.env.example` to `.env.local` and fill in your credentials.
   ```bash
   cp .env.example .env.local
   ```

4. **Run the development server**:
   ```bash
   pnpm dev
   ```

5. **Build for production**:
   ```bash
   pnpm build
   ```
